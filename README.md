---

## 🏗 Architecture

Built using:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Static Export (`output: "export"`)
- GitHub Pages Deployment

The site is fully static for performance, SEO, and long-term maintainability.

---

## 📂 Sections

- `/` – Portfolio
- `/180days` – 180 Days OfflineLifeArchive

Each section uses route-based layout theming.

---

## 🚀 Development

Install dependencies:

```bash
npm install
````

Run locally:

```bash
npm run dev
```

Build static export:

```bash
npm run build
```

---

## 🔐 Notes

* No backend.
* No runtime API dependencies.
* Fully static architecture.
* Optimized for SEO and long-term content publishing.
